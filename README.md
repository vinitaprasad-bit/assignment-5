💸 Project Summary – “WealthZen” (Smart Money Manager Web App)

🌐 Project Overview



WealthZen 💸 is a modern web-based financial management application that allows users to:



Create and manage their account

Track income and expenses

View real-time stock market trends 📊

Get basic investment guidance



The main objective of this project is to help users manage their finances efficiently while also introducing them to investment opportunities in a simple and user-friendly way.



🎨 Technologies Used

🧱 HTML (Structure)



HTML is used to build the overall structure of the application:



Login Page (username, password, bank selection)

Dashboard (welcome screen with animation)

Main Application Page

Transaction table

Forms, buttons, input fields

Iframe for embedding stock graphs



👉 HTML provides the basic layout and structure of the app.



🎨 CSS (Styling \& UI Design)



CSS is used to make the application visually attractive and modern:



Dark gradient background for premium feel

Glassmorphism effect (blur + transparency)

Styled buttons with hover effects

Card layout for income and expense

Table styling

Animations (floating icons 💰📊)

Responsive design using media queries



👉 CSS enhances the user interface and visual experience.



⚙️ JavaScript (Functionality \& Logic)



JavaScript is the core of the application and handles all dynamic behavior:



🔐 Authentication System

User signup (data stored in localStorage)

Login validation

Bank selection stored with user

💰 Transaction Management

Add income or expense

Delete individual transactions

Clear all transactions

Search transactions by category

📊 Calculations

Total Income

Total Expense

Balance = Income - Expense

Real-time updates

🔒 Security Feature

Password verification required before adding a transaction

📈 Stock Market Integration

Dynamic stock graph using TradingView

Option to switch between different stocks

External stock website access



👉 JavaScript makes the app interactive, dynamic, and functional.



📱 Responsiveness



The application is fully responsive:



Flexbox is used for layout management

Media queries adjust layout for smaller screens

Mobile-friendly design

Elements resize automatically



👉 Ensures smooth experience on mobile, tablet, and desktop devices.



🧩 Feature-wise Explanation

🔐 Login System



Users enter their username, password, and select their bank.

The data is stored in localStorage, and login is validated before access is granted.



👋 Dashboard Page



Displays a personalized welcome message with a soft note and animated elements to enhance user experience.



💰 Transaction System



Users can:



Add income or expense

Store data dynamically

View transactions in a table

📊 Financial Calculations



The app automatically calculates:



Total Income

Total Expense

Remaining Balance



These values update in real-time whenever a transaction is added or removed.



🔍 Search Feature



Allows users to filter transactions based on category for easy access.



🧹 Delete \& Clear Options

Delete individual transactions

Clear all transaction records

🔒 Security Layer



Before adding a transaction, the user must re-enter their password, ensuring basic security protection.



📈 Stock Market Feature

Displays a live stock graph using TradingView

Users can switch between different stocks

External stock platforms can be opened for detailed analysis

💡 Investment Suggestion



A smart guiding line is displayed:



“Want to invest your funds wisely? Explore real-time market trends and make smarter financial decisions.”



This encourages users to explore investment opportunities.



🚀 Conclusion



WealthZen is a complete frontend-based fintech web application that:



✔ Provides secure login functionality

✔ Helps users manage finances effectively

✔ Offers real-time stock insights

✔ Is responsive and user-friendly



This project demonstrates:



Strong understanding of HTML, CSS, and JavaScript

DOM manipulation

Local storage handling

UI/UX design principles

Real-world application integration
